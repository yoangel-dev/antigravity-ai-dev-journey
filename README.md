# 🌌 Antigravity AI Dev Journey
### 🚀 Laboratorio de Proyectos Web Generados con IA — React • TypeScript • Tailwind • Vite

Bienvenido a **Antigravity AI Dev Journey**, un repositorio profesional que recopila proyectos web modernos desarrollados mediante Google AI Studio (Antigravity) y optimizados manualmente con los más altos estándares de ingeniería frontend.

Este espacio funciona como un laboratorio de innovación, donde cada proyecto demuestra cómo combinar:
* **Generación asistida por IA:** Aceleración del desarrollo y prototipado inteligente.
* **Refinamiento técnico profesional:** Optimización de código, accesibilidad y rendimiento.
* **Buenas prácticas de arquitectura frontend:** Estructuras limpias, modulares y escalables.
* **Diseño moderno y accesible:** Interfaces pulidas, responsivas y centradas en el usuario.
* **Documentación clara y reproducible:** Procesos transparentes de instalación y extensión.

---

## 🧭 Propósito del Repositorio

Este repositorio tiene tres objetivos principales:

1. **Centralizar proyectos generados con IA:** Cada carpeta contiene un proyecto completo, funcional y documentado, listo para ejecutar, analizar o extender.
2. **Mostrar un flujo de trabajo profesional:** Demostrar el ciclo completo de desarrollo: desde el *prompt* inicial hasta la arquitectura modular, commits descriptivos, tipado estricto con TypeScript y documentación técnica robusta.
3. **Construir un portfolio técnico sólido:** Servir como base pública para demostrar el dominio de tecnologías modernas, la capacidad de integrar IA de vanguardia en el flujo de trabajo diario y la evolución continua como desarrollador.

---

## 🛠️ Tecnologías Core (Stack Tecnológico)

Todos los proyectos están construidos con tecnologías de última generación para garantizar rendimiento, mantenibilidad y una experiencia de usuario excepcional:

| Categoría | Tecnología | Descripción |
| :--- | :--- | :--- |
| **Framework** | React 19 | Arquitectura declarativa de componentes y manejo eficiente del DOM. |
| **Lenguaje** | TypeScript | Tipado estricto para un código seguro, auto-documentado y libre de errores en runtime. |
| **Estilos** | Tailwind CSS v4 | Estilizado moderno basado en utilidades con excelente rendimiento. |
| **Build Tool** | Vite | Entorno de desarrollo ultrarrápido y bundling optimizado para producción. |
| **Animaciones**| Framer Motion | Transiciones fluidas e interacciones dinámicas de alta fidelidad. |
| **Iconografía**| Lucide React | Set de iconos vectoriales limpios, consistentes y optimizados para SVG. |
| **Estado** | Context API / Custom Hooks | Gestión del estado global y lógica de negocio desacoplada de la UI. |
| **IA Core** | Google AI Studio (Antigravity) | Motor de IA generativa utilizado para la concepción y co-creación del código base. |

## 📁 Estructura del Repositorio

El repositorio organiza cada proyecto de forma independiente para facilitar su exploración y despliegue:

```bash
antigravity-ai-dev-journey/
│
├── 01-Tareas-pendientes/          # Proyecto Destacado: Crimson Flow
│   ├── src/                       # Código fuente de la aplicación
│   ├── assets/                    # Recursos estáticos (imágenes, fuentes)
│   ├── components/                # Componentes atómicos y moleculares reutilizables
│   ├── index.html                 # Punto de entrada de la aplicación
│   ├── package.json               # Dependencias y scripts del proyecto
│   ├── vite.config.ts             # Configuración del empaquetador Vite
│   └── README.md                  # 📄 Documentación técnica detallada del proyecto
│
├── 02-<proyecto-futuro>/          # Próximas innovaciones del laboratorio
│   └── README.md
│
└── README.md                      # 📜 Documentación principal (Este archivo)

Cada carpeta de proyecto incluye su propio entorno de dependencias y configuración de compilación, asegurando que sean módulos 100% autónomos.

🩸 Proyecto Destacado
01 — Crimson Flow: Gestor de Tareas Personales
Crimson Flow es una aplicación web premium diseñada para la gestión de tareas diarias, que combina una estética visualmente sofisticada con animaciones fluidas y una experiencia de usuario impecable.

✨ Características clave:

Modal dinámico: Flujo intuitivo para crear, editar y categorizar tareas en tiempo real.

Modo Claro/Oscuro: Transición de color suave mediante variables nativas y Tailwind.

Filtros inteligentes: Organización avanzada por estado de cumplimiento y fechas límite.

Perfil editable: Panel de usuario dinámico con actualización de avatar en tiempo real.

Persistencia local: Integración nativa con localStorage para salvaguardar los datos.

Accesibilidad (a11y): Navegación optimizada mediante teclado y marcado semántico.

Animaciones interactivas: Microinteracciones fluidas impulsadas por Framer Motion.

👉 Documentación completa del proyecto: /01-Tareas-pendientes/README.md

🚀 Cómo Ejecutar Cualquier Proyecto
Sigue estos sencillos pasos para levantar cualquier entorno del laboratorio de forma local:

Clonar el repositorio:

Bash
git clone [https://github.com/yoangel-dev/antigravity-ai-dev-journey](https://github.com/yoangel-dev/antigravity-ai-dev-journey)
Navegar a la carpeta del proyecto que deseas probar:

Bash
cd antigravity-ai-dev-journey/01-Tareas-pendientes
Instalar las dependencias del proyecto:

Bash
npm install
Iniciar el servidor de desarrollo local:

Bash
npm run dev
Abre el enlace generado en tu consola (usualmente http://localhost:5173) para ver la aplicación en acción.
