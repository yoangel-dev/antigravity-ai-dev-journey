## AGENTS

Proyecto: Generador de contraseñas (utilidad para publicar en internet) 

Rol del agente: Desarrollador web experto con 12 años de experiencia.

Objetivo: Crear una aplicación web que genere contraseñas aleatorias y seguras.

Funcionalidades de la aplicación:

- Seleccionar longitud
- Activar / Desactivar:
    - Mayúsculas
    - Minúsculas
    - Números
    - Caracteres especiales
- Generar contraeña aleatoria
- Copiar contraeña al portapapeles
- Indicador de fortaleza de la contraseña
- Mostrar / ocultar contraseña


Stack de tecnología:

- HTML5
- CSS3(sin frameworks)
- JavaScript (Vanilla, sin frameworks)

Preferencias de diseño:

- Minimalista
- Moderno
- Limpio
- Intuitivo
- Responsivo
- Adaptable a móviles y escritorio
- Claro

Preferencias de estilos:

- Colores azules y frios
- Uso de medidas con rem, usando un font-size base de 10px
- Uso de HTML5 y CSS3 nativo (sin tailwind, ni frameworks)
- Usa buenas practicas de maquetación css y si es    necesario usa flexbox y css grid layout.
- Que la app esté en idioma Español.
- Usa el font-size en rem, usando un font-size base de 1rem = 10px.
- Que la WebApp sea responsiva y adaptable a cualquier dispositivo.
- Que la WebApp sea accesible cumpliendo con las normas WCAG.

Preferencias de código:

- No añadas dependencias externas.
- HTML debe ser semantico
- No uses alert, confirm, prompt, todo el feedback debe ser visual en el DOM
- No uses innerHTML, todo el contenido debe ser insertado con appendChild, o previamente creado un elemento con document.createElement.
- Cuidado con olvidar prevenir el default de los eventos en submits o clicks
- Prioriza el codigo legible y mantenible.
- Comenta el codigo.
- Prioriza que el codigo sea sencillo de leer y entender.
- Si el agente duda, que revise las especificaciones del proyecto y si no que pregunte al usuario.

Estructura de archivos:

- carpeta(assets)
    -carpeta(css)
    -carpeta(js)
    -carpeta(images)
- index.html
- AGENTS.md
- README.md
