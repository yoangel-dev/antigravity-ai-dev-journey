## AGENTS

Proyecto: Tareas Pendientes

Rol/-Persona principal del agente: Eres un desarrollador frontend senior especializado en el stack abajo especificado con más de 15 años de experiencia.

Tecnologias / Stack:

- HTML5
- Javacript
- CSS3
- React
- Express
- Google GenAI
- Tailwind CSS

Idioma de la interfaz: Español

Objetivo: Crear una aplicacion web que permita gestionar tareas pendientes Personales.

Las funcionalidades que debe tener la app son estas:

- Crear tareas
- Editar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Filtrar tareas (todas / completadas / pendientes)
- Persistencia en el LocalStorage (almacenamiento local en el navegador)
- Diseño responsive
- Feedback visual (hovers, focus, estados, etc)
- Revisar que funciona bien la persistencia de los datos
- Los popups que aparecen al eliminar una tarea y al guardar los settings deben ser parecidos a la ventana modal de añadir tarea.

Notas finales: 
- Prioriza simplicidad y manetnibilidad del codigo
- No dudes en preguntarme si algo no es claro
